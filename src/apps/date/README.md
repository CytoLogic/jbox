# date

Display the current date and time.

## Synopsis

```
date [-h]
```

## Description

Display the current date and time in the default format.

## Options

| Option | Description |
|--------|-------------|
| `-h, --help` | Display help and exit |

## Examples

Display current date and time:
```
date
```

Example output:
```
Sat Dec 14 10:30:45 2024
```

## Exit Status

- `0` - Success
